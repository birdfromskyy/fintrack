# Fintrack - Система учёта личных финансов

## Описание

Fintrack - это веб-приложение для учёта личных финансов, построенное на микросервисной архитектуре с использованием современных технологий.

## Технологический стек

### Backend

- **Go + Gin** - веб-фреймворк для API
- **PostgreSQL** - основная база данных
- **ClickHouse** - аналитика и логирование
- **Redis** - кеширование и сессии
- **JWT** - аутентификация

### Frontend

- **React** - UI библиотека
- **Redux Toolkit** - управление состоянием
- **Material-UI** - компоненты интерфейса
- **Chart.js & Recharts** - визуализация данных

### Инфраструктура

- **Docker & Docker Compose** - контейнеризация
- **Grafana** - мониторинг и дашборды

## Архитектура

Приложение состоит из трёх микросервисов:

1. **Auth Service (порт 8081)** - регистрация, аутентификация, управление пользователями
2. **API Service (порт 8082)** - основная бизнес-логика (транзакции, счета, категории)
3. **Analytics Service (порт 8083)** - аналитика, отчёты, экспорт данных

## Основные функции

- ✅ Регистрация с подтверждением email
- ✅ Управление счетами
- ✅ Категоризация доходов и расходов
- ✅ Добавление транзакций
- ✅ Аналитика и статистика
- ✅ Прогнозирование
- ✅ Экспорт данных в CSV
- ✅ Дашборды в Grafana
